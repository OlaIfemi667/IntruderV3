What is Intruder?

Intruder is a tool that automate footprinting phase in pentesting process.

It is code in python and uses tools like nmap, exploit-db, zap, whois

Key Penetration Testing Phases

1. Footprinting & Reconnaissance

The initial phase of gathering information about the target system, including:

  • Network enumeration
  • Domain information gathering
  • WHOIS lookups
  • DNS enumeration
  • Port scanning
  • Service identification

2. Scanning

Active probing of target systems to identify:

  • Open ports and services
  • Operating system detection
  • Vulnerability scanning
  • Network mapping

3. Enumeration

Detailed information gathering about:

  • User accounts
  • Machine names
  • Network resources
  • Services and applications

Intruder is basically a kind of wrapper, it uses nmap, whois, exploit-db and ZAP api

to automate some pentests tasks and an LLM for best reports and for chat
